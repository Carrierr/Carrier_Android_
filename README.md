## 주요 기능

### 1. 방문한 장소의 사진과 느낌을 적어보세요.
>작성할 때 마다 새로운 우표를 받을 수 있습니다.

### 2. 나의 팁탭을 타인과 공유해보세요.
>같은 장소에서 타인은 어떻게 느꼈을까요?

### 3. 쌓여가는 나의 발자취를 한눈에 확인해보세요.
